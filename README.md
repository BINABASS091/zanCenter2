# Zanzibar.Center Experience Hub

A global interactive gaming and learning platform for children aged 5-16. Play mini-games, build islands, explore zones, and learn React web development.

## Features

- **Game Hub** - Play Quiz Arena, Tap Quest, and other mini-games
- **Island Canvas** - Drag-and-drop island building simulation
- **React Code Academy** - Learn web development step-by-step
- **Zone Explorer** - Discover hidden island regions with rewards
- **AI Coach** - Contextual learning guidance
- **Portfolio Showcase** - Publish the websites you build
- **Parent & Teacher Portals** - Progress tracking dashboards

## Zanzibar Ecosystem

This Experience Hub is part of the Zanzibar.Center ecosystem:

- [Zanzibar.Center](https://zanzibar.center) - Children's Learning Village in Paje
- [Zanzibar.Camp](https://zanzibar.camp) - Children's Holiday Camps
- [Calendar](https://calendar.zanzibar.center) - Event Schedule
